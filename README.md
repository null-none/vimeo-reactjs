# youtube-react
React component acting as Vimeo iFrame.

## Install

```bash
npm install vimeo-reactjs
```

## Example

```javascript
import Vimeo from 'vimeo-reactjs';

...
<Vimeo video="76979871" height="315px" width="540px" />
...

```
